# ✅ Habit tracker
Habit tracker app made with React

# 🤖 Technologies
- React
- JavaScript
- CSS

# 🔥 Features
- Add habit
- Delete habit
- Edit habit
- Completed habits progress bar
- Light and dark mode
- Habits saved in localStorage
- Habits get unchecked every day
